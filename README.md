## UI helpers

* Use the `btn` class on links and buttons to make them large, colored, and responsive.
* Buttons and anchors automatically expand to full width on small screens.
* The entire UI now uses a dark, Spotify‑inspired theme (dark backgrounds, green accents, clean cards, bottom navigation) to feel like a mobile music app.
* A lightweight modal system is included. Call `showModal({title:'Title', body:'HTML or text'})` from any page. The overlay and markup are injected via `js/app.js`.
* Close the modal by clicking the × or the background, or call `hideModal()` programmatically.
